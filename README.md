# 💸 Smart Expense Tracker

A modern full-stack expense tracking web application built using Flask, featuring a clean dashboard UI, category-based analytics, and interactive data visualization.

---

## 🚀 Features

- 📊 Dashboard with total spending overview
- 🧾 Add & delete expenses
- 🗂 Category-based expense tracking
- 📅 Date-wise expense management
- 📈 Interactive charts using Chart.js
- 🎨 Clean and minimal UI design

---

## 🛠 Tech Stack

- Python (Flask)
- SQLite
- HTML, CSS
- Chart.js
